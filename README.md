# 招商银行pdf流水提取分析工具

# ✨ 使用方法

- 直接启动选择pdf文件即可

# 📦 打包方法

- 运行 files/package.bat 
- files/package/main 文件夹即是打包结果，直接运行main.exe即可。

# ⚙️ 设置项说明

| 配置项       | 说明                                    |
|-----------|---------------------------------------|
| save_path | 导出文件的存储目录，可以写绝对路径或相对路径                |
| area      | pdf边距定义，[top, left, bottom, right] ， 一般不需要更改   |
| area1     | pdf第一页边距定义，[top, left, bottom, right]， 一般不需要更改 |
